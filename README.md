AdAndCrapBlock
==============

This tool downloads the AdAway  host-files and merge it into the file /etc/hosts

<h3>Ubuntu</h3>
<pre>
sudo apt-add-repository ppa:memoryleakx/adandcrapblock 
sudo apt-get update 
sudo apt-get install adandcrapblock
</pre>

<h3>Arch Linux</h3>
<pre>
yaourt -S adandcrapblock
</pre>
